Triển khai tự động đếm "Số lượt tải" (Cloudflare Worker + KV)

Nếu GitHub Actions báo lỗi (npx exit 1):
  A) Mở tab job "deploy" → xem log. Thường là: chưa thay REPLACE_ trong wrangler.toml, hoặc thiếu Secrets.
  B) Làm đúng thứ tự bên dưới rồi "Re-run jobs".

Bước 1 — Tạo KV và điền wrangler.toml (máy local, đã cài Node.js)
  cd workers/install-counter
  npm install
  npx wrangler login
  npx wrangler kv namespace create INSTALL_KV
  Copy dòng id (UUID) — dán thay thế REPLACE_WITH_KV_NAMESPACE_ID trong wrangler.toml
  Commit + push file wrangler.toml lên GitHub.

Bước 2 — GitHub Secrets (repo jaboz.com)
  Settings → Secrets and variables → Actions → New repository secret
  - CLOUDFLARE_ACCOUNT_ID = copy từ Cloudflare Dashboard (cột phải trang tổng quan)
  - CLOUDFLARE_API_TOKEN = tạo token (Workers KV Storage Edit + Workers Scripts Edit, hoặc tương đương)

Bước 3 — Chạy lại workflow
  Tab Actions → workflow failed → Re-run all jobs

Bước 4 — Deploy tay (nếu không dùng CI)
  npx wrangler deploy
  URL: https://hoctiengviet-install-count.<subdomain>.workers.dev

Bước 5 — Gắn URL vào app Android
  gradle.properties: hoctiengviet.installStatsBaseUrl=https://... (không / cuối)
  Build lại APK.

Tùy chọn — jaboz.com không đổi app
  Cloudflare → Workers Routes: /install-stats.json* và /install-count-register* → worker này.
